# PRODUCT.md

## App Name
Temporary working name: TripSync

## Core Idea
A mobile app for friends traveling in separate cars to the same destination, allowing them to see each other’s live locations during a shared trip.

## Target Users
Small groups of friends or family.
Usually 2 to 10 people.
Usually short or medium-length shared trips.

## Core Problem
Google Maps and similar apps do not provide a simple trip-first shared live location experience optimized for multiple friends traveling together in separate vehicles.

## MVP
- Create trip
- Join trip
- View shared live map
- Share location during trip
- Stop sharing
- End trip

## Non-Goals
- public social network
- family surveillance app
- navigation replacement
- monetization-first app
- large-scale platform
